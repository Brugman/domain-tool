# Domain Tool

## Requirements

- Apache (any webserver)
- PHP
- Composer
- npm

## Installation

1. `composer i`
1. `npm i`
1. Copy `.env.example` to `.env` and configure it.
1. Map `public_html` to a (sub)domain.
